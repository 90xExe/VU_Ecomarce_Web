Eco_Website - 90N.GameShop Wallet + bKash Pending System

Install:
1. Extract ZIP.
2. Copy Eco_Website folder to C:\xampp\htdocs\
3. Start Apache from XAMPP.
4. Open: http://localhost/Eco_Website/

New in this version:
- User profile has more details:
  name, email, phone, default Free Fire UID, address/note
- Add Money no longer instantly adds balance
- Add Money opens bKash style payment page
- User enters transaction ID and verifies
- Add money request is saved as pending
- Profile shows Add Money Requests table
- Wallet balance only changes later when admin approves
- Topup order still uses wallet balance only
